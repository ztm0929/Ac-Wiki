# 关于截图

比较常用的两种截图方式

## 使用软件内置功能

这里仅展示默认快捷键

- QQ 截图：`ctrl + alt +a`
- 微信截图：按下键盘上的`Alt + A`就可以实现微信截图

## Windows 系统截图

- 全屏截图：`PrtSc`，按下后会写入剪贴板，可直接在 画图、PhoneShop、GIMP、QQ、微信等软件中直接粘贴
- 区域截图：`Win+Shift+S`
